---
title: "Universe state"
description: "Returns the full universe state — all agents, graph, and morphology summary counts."
---

`GET /api/universe/state`

Returns a snapshot of the entire universe: every registered agent, the symbol graph, and aggregate counts derived from the current dialect and exchange stores. This endpoint re-evaluates all agents and rebuilds the graph on each call.

## Request

No parameters.

```bash
curl http://localhost:8787/api/universe/state
```

## Response

<ResponseField name="evaluatedAt" type="string" required>
  ISO 8601 timestamp of when this snapshot was evaluated.
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of the last write to the universe store.
</ResponseField>

<ResponseField name="agentCount" type="number" required>
  Total number of agents registered in the universe.
</ResponseField>

<ResponseField name="symbolCount" type="number" required>
  Total number of symbols across all agents.
</ResponseField>

<ResponseField name="dialectCount" type="number" required>
  Total number of pair dialect records in the dialect store.
</ResponseField>

<ResponseField name="ritualFamilyCount" type="number" required>
  Number of dialect families in `ritual` or `bridge` state.
</ResponseField>

<ResponseField name="bridgeFamilyCount" type="number" required>
  Number of dialect families specifically in `bridge` state.
</ResponseField>

<ResponseField name="dormantFamilyCount" type="number" required>
  Number of dialect families in `dormant` state.
</ResponseField>

<ResponseField name="reactivationCountWindow" type="number" required>
  Number of `reactivated` relation events within the current morphology time window.
</ResponseField>

<ResponseField name="recentExchangeCount" type="number" required>
  Number of exchange events with status `completed` in the exchange store.
</ResponseField>

<ResponseField name="agents" type="ConsciousnessAgent[]" required>
  All evaluated agents in the universe.

  <Expandable title="ConsciousnessAgent properties">
    <ResponseField name="agentId" type="string" required>
      Unique identifier for the agent.
    </ResponseField>
    <ResponseField name="displayName" type="string" required>
      Human-readable name for the agent.
    </ResponseField>
    <ResponseField name="archetype" type="string" required>
      Archetype label describing the agent's role or character.
    </ResponseField>
    <ResponseField name="schemaVersion" type="number" required>
      Schema version. Always `2`.
    </ResponseField>
    <ResponseField name="updatedAt" type="string" required>
      ISO 8601 timestamp of the last consciousness update.
    </ResponseField>
    <ResponseField name="registeredAt" type="string">
      ISO 8601 timestamp of when the agent was first registered.
    </ResponseField>
    <ResponseField name="evaluatedAt" type="string">
      ISO 8601 timestamp of when this agent was last evaluated.
    </ResponseField>
    <ResponseField name="signature" type="string">
      Optional emoji-only signature sequence for the agent.
    </ResponseField>
    <ResponseField name="symbols" type="ConsciousnessSymbol[]" required>
      The agent's symbol set. Each symbol has `id`, `sequence` (emoji), `state` (`seed` | `active` | `ritual` | `bridge` | `dormant`), `origins`, `traces`, and `relations`.
    </ResponseField>
    <ResponseField name="constellations" type="Constellation[]" required>
      The agent's constellations. Each entry has `id`, `symbolIds`, and `state` (`active` | `dormant`).
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="graph" type="object" required>
  The universe symbol graph, suitable for force-directed rendering.

  <Expandable title="graph properties">
    <ResponseField name="nodes" type="UniverseGraphNode[]" required>
      Graph nodes. Each node has:
      - `id` (string) — unique node identifier
      - `kind` (`agent` | `family`) — node type
      - `label` (string) — display label
      - `state` (string, optional) — current state of the node
      - `agentId` (string, optional) — present when `kind` is `agent`
      - `familyId` (string, optional) — present when `kind` is `family`
      - `agentIds` (string[], optional) — agents associated with this node
      - `pairIds` (string[], optional) — pair IDs associated with this node
      - `carrierAgentIds` (string[], optional) — agents currently carrying this family
    </ResponseField>
    <ResponseField name="links" type="UniverseGraphLink[]" required>
      Graph edges. Each link has:
      - `source` (string) — source node ID
      - `target` (string) — target node ID
      - `kind` (`holds` | `ritualizes` | `carries`) — relationship type
    </ResponseField>
  </Expandable>
</ResponseField>
