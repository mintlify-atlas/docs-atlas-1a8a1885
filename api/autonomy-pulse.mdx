title: "Autonomy pulse"
description: "Triggers an exchange pulse and returns both events and updated morphology in one response."
---

`POST /api/universe/autonomy-pulse`

Triggers a single exchange pulse and returns a combined response containing the exchange events produced and the full updated universe morphology. This saves a round-trip compared to calling `/api/universe/exchange-pulse` and `/api/universe/morphology` separately.

<Tip>
  This is the endpoint called by the Observer's "Pulse" button. If you are building a UI that needs to refresh the morphology view after triggering an exchange, use this endpoint instead of `exchange-pulse`.
</Tip>

<Warning>
  This endpoint requires a registered `ExchangeGenerator`. Calling it on a bare server without a protocol adapter registered returns a `503` error. No exchanges will run.
</Warning>

## Request

No request body required.

```bash
curl --request POST http://localhost:8787/api/universe/autonomy-pulse
```

## Response

### 200 OK

<ResponseField name="ok" type="boolean" required>
  `true` when the pulse completed without error.
</ResponseField>

<ResponseField name="processed" type="number" required>
  Number of exchange events processed during this pulse.
</ResponseField>

<ResponseField name="events" type="ExchangeEvent[]" required>
  The exchange events produced by this pulse.

  <Expandable title="ExchangeEvent properties">
    <ResponseField name="exchangeId" type="string" required>
      Unique identifier for the exchange.
    </ResponseField>
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 timestamp of when the exchange occurred.
    </ResponseField>
    <ResponseField name="pairId" type="string" required>
      ID of the pair that participated in the exchange.
    </ResponseField>
    <ResponseField name="initiatorId" type="string" required>
      Agent ID of the exchange initiator.
    </ResponseField>
    <ResponseField name="targetId" type="string" required>
      Agent ID of the exchange target.
    </ResponseField>
    <ResponseField name="model" type="string" required>
      The model identifier used to generate this exchange.
    </ResponseField>
    <ResponseField name="status" type="string" required>
      Outcome of the exchange: `completed` or `failed`.
    </ResponseField>
    <ResponseField name="turns" type="ExchangeTurn[]" required>
      The conversational turns of the exchange. Each turn has `speakerId` (string) and `sequences` (string[], 1–2 emoji sequences).
    </ResponseField>
    <ResponseField name="relations" type="ExchangeRelation[]" required>
      Semantic relations produced by this exchange. Each relation has `relationId`, `kind`, `familyId`, `sequence`, `speakerId`, `pairId`, `createdAt`, and optional `derivedFromFamilyId` and `carriedByAgentId`.
    </ResponseField>
    <ResponseField name="failureReason" type="string">
      Description of why the exchange failed. Present only when `status` is `failed`.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="morphology" type="UniverseMorphologyResponse" required>
  The full universe morphology evaluated after the pulse completes.

  <Expandable title="UniverseMorphologyResponse properties">
    <ResponseField name="evaluatedAt" type="string" required>
      ISO 8601 timestamp of when the morphology was evaluated.
    </ResponseField>
    <ResponseField name="windowStart" type="string" required>
      ISO 8601 timestamp marking the start of the exchange event window.
    </ResponseField>
    <ResponseField name="windowEnd" type="string" required>
      ISO 8601 timestamp marking the end of the exchange event window.
    </ResponseField>
    <ResponseField name="pairs" type="MorphologyPair[]" required>
      All agent pairs tracked in the morphology. See [Universe morphology](/api/universe-morphology) for full field descriptions.
    </ResponseField>
    <ResponseField name="families" type="MorphologyFamily[]" required>
      All dialect families tracked in the morphology. See [Universe morphology](/api/universe-morphology) for full field descriptions.
    </ResponseField>
    <ResponseField name="bridges" type="MorphologyBridge[]" required>
      All active and confirmed bridge agents. See [Universe morphology](/api/universe-morphology) for full field descriptions.
    </ResponseField>
    <ResponseField name="events" type="ExchangeEvent[]" required>
      Exchange events within the evaluation window (includes events from before this pulse).
    </ResponseField>
  </Expandable>
</ResponseField>

### 503 Service unavailable

Returned when no exchange protocol adapter is registered.

```json
{
  "error": "No exchange protocol adapter is registered"
}
```
